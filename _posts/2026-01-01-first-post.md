## 缘起

这学期开始系统学习人工智能相关课程，想找一个地方持续记录学习过程中的收获和思考。于是决定用 GitHub Pages + Jekyll 搭建这个博客。

## 为什么选 Jekyll

1. **免费托管**：GitHub Pages 完全免费，不需要自己买服务器
2. **Markdown 写作**：用熟悉的 Markdown 格式写文章，专注内容而非排版
3. **版本管理**：所有文章都在 Git 仓库里，修改历史一目了然
4. **自动构建**：推送即发布，不需要手动部署

## 博客的计划

后续会在这里持续更新以下内容：

- 课程学习笔记（机器学习、深度学习、计算机视觉）
- 项目实践记录和踩坑经验
- 读论文的总结和思考
- 技术工具的配置和使用技巧

希望这个博客能见证我的成长。保持学习，保持记录。
