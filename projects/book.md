## 项目概述

大二《Python 程序设计》期末大作业。基于**协同过滤（Collaborative Filtering）**算法实现个性化图书推荐系统。用户评分后自动推荐相似用户喜欢的书籍，以及和已评高分图书相似的书籍。

| 指标 | 数值 |
|------|------|
| 图书数据 | **5000+** 本 |
| 预测误差 | **RMSE 0.87** |
| Top-10 命中率 | **78%** |

## 算法设计

采用混合推荐策略，结合 UserCF 和 ItemCF：

| 维度 | UserCF | ItemCF |
|------|--------|--------|
| 相似度 | 余弦相似度 | 皮尔逊相关系数 |
| K 近邻 | Top-30 | Top-20 |
| 稀疏处理 | 均值填充 | SVD 降维 |

```
混合策略（权重通过网格搜索优化）
final_score = 0.45 × user_cf_pred + 0.55 × item_cf_pred
```

## Web 功能

1. **用户注册/登录**：Flask-Login 管理会话，bcrypt 密码加密
2. **图书搜索**：按书名、作者、分类模糊搜索
3. **评分系统**：1~5 星评分，实时更新均分和用户画像
4. **个性化推荐**：首页"为你推荐" + 推荐理由标注
5. **热门榜单**：全校用户评分数据实时排行

## 数据来源

图书元数据来自 Goodreads Books Dataset（5000+ 本中英文图书标题、作者、分类、简介）。评分数据通过模拟生成（300 虚拟用户）。

## 收获

掌握协同过滤、矩阵分解和冷启动应对策略。体会到数据稀疏性是推荐系统的最大挑战——稀疏评分矩阵严重影响相似度计算可靠性。
